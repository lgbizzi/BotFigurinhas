# Política de Privacidade — Bot Figurinhas Copa 2026

**Última atualização:** 24 de maio de 2026

---

## 1. Quem somos

O **Bot Figurinhas Copa 2026** é um serviço pessoal e não comercial, operado por uso privado, que permite aos usuários do Telegram controlar sua coleção de figurinhas do álbum oficial da Copa do Mundo 2026.

---

## 2. Dados coletados

Coletamos o **mínimo necessário** para identificar cada usuário de forma única e associar seu álbum ao seu perfil:

| Dado | Origem | Finalidade |
|---|---|---|
| **Telegram User ID** | Fornecido automaticamente pelo Telegram a cada mensagem | Identificar o usuário de forma única e isolar seu álbum dos demais |
| **Telegram Username** (`@usuario`) | Fornecido automaticamente pelo Telegram | Registro interno de logs para diagnóstico de erros |

Não coletamos nome, e-mail, telefone, localização, dados de pagamento ou qualquer outra informação pessoal.

---

## 3. Como os dados são usados

Os dados coletados são usados exclusivamente para:

- Criar e manter o álbum individual de cada usuário no banco de dados
- Registrar o histórico de movimentações (adições e remoções de figurinhas) vinculadas ao usuário
- Identificar a origem de erros nos logs internos da aplicação

Os dados **não são** usados para fins comerciais, publicidade, análise de comportamento ou compartilhados com terceiros.

---

## 4. Armazenamento e segurança

- Os dados são armazenados em um banco de dados PostgreSQL hospedado em servidor privado
- O acesso ao banco é restrito à aplicação, por meio de um usuário com permissões mínimas
- Nenhuma informação é transmitida a serviços externos além da API do Telegram (necessária para o funcionamento do bot)
- O token do bot e as credenciais do banco nunca são armazenados no código-fonte

---

## 5. Retenção de dados

Os dados são mantidos enquanto o usuário utilizar o bot. Não há exclusão automática por inatividade.

---

## 6. Direitos do usuário

Você pode solicitar a qualquer momento:

- **Consulta:** saber quais dados estão armazenados sobre você
- **Exclusão:** remover completamente seu álbum e histórico de movimentações do banco de dados

Para exercer esses direitos, entre em contato diretamente com o administrador do bot via Telegram.

---

## 7. Crianças

Este bot não é direcionado a crianças. Não coletamos intencionalmente dados de menores de idade.

---

## 8. Alterações nesta política

Eventuais atualizações serão publicadas neste arquivo. A data de "Última atualização" no topo do documento será revisada a cada alteração.

---

## 9. Contato

Dúvidas ou solicitações relacionadas a esta política podem ser enviadas diretamente ao administrador via Telegram.
